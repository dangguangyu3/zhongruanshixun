# 数码手机商城项目

## 项目简介
这是一个基于Java纯控制台的数码手机商城项目，采用Maven构建，使用MySQL数据库存储数据。项目实现了普通用户和管理员两套完整的功能体系。

## 技术栈
- Java 1.8
- Maven 3.8+
- MySQL 8.0

## 项目结构
```
digital-phone-mall/
├── src/
│   ├── main/
│   │   ├── java/com/digitalmall/
│   │   │   ├── entity/         # 实体类
│   │   │   ├── dao/            # 数据访问层
│   │   │   │   └── impl/       # DAO实现类
│   │   │   ├── service/        # 业务逻辑层
│   │   │   ├── util/           # 工具类
│   │   │   └── Main.java       # 主启动类
│   │   └── resources/
│   │       └── init.sql        # 数据库初始化脚本
│   └── test/
└── pom.xml                     # Maven配置文件
```

## 数据库配置
1. 确保MySQL 8.0已安装并运行
2. 数据库用户名：root
3. 数据库密码：123456
4. 运行`src/main/resources/init.sql`脚本初始化数据库

## 项目导入
1. 打开IntelliJ IDEA
2. 选择"File" -> "Open"
3. 选择项目根目录`digital-phone-mall`
4. 等待Maven依赖下载完成

## 项目运行
1. 确保数据库已初始化
2. 运行`Main.java`类的`main`方法
3. 进入控制台菜单交互界面

## 功能说明

### 普通用户功能
1. **注册**：创建新用户账号
2. **登录**：使用用户名和密码登录
3. **浏览全部手机商品**：查看所有上架的手机商品
4. **查看商品详情**：查看商品的详细信息
5. **按品牌/价格筛选搜索手机**：根据品牌或价格范围搜索商品
6. **购物车管理**：添加商品、修改数量、删除商品、结算
7. **查看历史订单**：查看用户的历史订单记录
8. **修改密码**：修改用户密码
9. **退出登录**：退出当前用户登录

### 管理员功能
1. **登录**：使用管理员账号登录
2. **商品管理**：新增手机、修改商品信息、上下架商品、删除商品、查询所有商品
3. **用户管理**：查看所有注册用户、管理用户账号状态
4. **订单管理**：查看平台所有订单、修改订单状态、按状态筛选查询订单
5. **商城数据统计**：销量统计、库存预警统计（功能开发中）
6. **修改密码**：修改管理员密码
7. **退出后台**：退出管理员登录
